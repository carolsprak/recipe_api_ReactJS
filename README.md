# Food Recipe Web Application

**Tecnologias:**

- React JS
- Food API
- Materialize CSS

**Como rodar este projeto?**

1. Clone este projeto
2. Entre na pasta onde está o projeto no terminal do VSCode

```
cd folder-name
```

3. Instale Dependencias usando `NPM`

```
npm install
```

4. Execute a aplicação

```
npm run start

```

5. Acesse a URL
 
- http://localhost:3000/
